canvas

canvas通过属性进行宽高设置，且不带单位；通过style设置的大小会使canvas画布失真

基本使用

    <canvas id="canvas" width="800" height="300"></canvas>
    <script>
        1.获取元素
            var canvas = document.querySelector('#canvas');
        2.获取绘制入口
            var hy = canvas.getContext('2d') // 意为画布是2d画布，webgl是3d画布，但有兼容性问题；
        3.指定起点和终点
            hy.moveTo(); // 取值为坐标不带单位
            hy.lineTo();
        4.描边
            hy.stroke();//  填充是使用 hy.fill()
    </script>

  • strokeStyle/fillStyle

      scrokeStyle与fillStyle()可以设置描边或填充的颜色；

  • beginPath()

      beginPath()是启用新路径，避免后面绘制描边的颜色与前面的路径发生冲突；

  • closePath()

      闭合路径；这个方法会通过绘制一条从当前点到开始点的直线来闭合图形。
  如果图形是已经闭合了的，即当前点为开始点，该函数什么也不做。

  • lineWidth

      设置线宽；

  • setLineDash()

      设置虚线，参数里面防一个数组,[];两个参数分别为实线的长度和虚线的长度；

  • getLineDash()

      获取虚线的值；

  • lineDashOffset()

      设置虚线与起点的距离；

  • clearRect()

      清空画布,clearRect(0,0,canvas.widht,canvas.height);从哪个坐标点开始清空，清空的范围是多少..

  • strokeRect()/fillRect()

      绘制矩形，strokeRect(x,y,w,h)；从哪个坐标点开始绘制，绘制的矩形的长度和宽度是多少?

  • arc()

      arc(x,y,radius,startAngle,endAngle,anticlockwise);画一个一x,y为圆心，以radius为半径的圆弧;
  从starAngle开始到engAngle结束，最后一个为绘制的方向，顺时针或逆时针方向；
