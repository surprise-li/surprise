git

Bash基本命令

  • pwd

      打印当前路径的名称(print working diretory)

  • ls

      列举当前目录(list)

  • cd

      切换目录(change diretory)

  • mkdir

      创建文件夹(make diretory)

  mkdir  app/aa  -p // app 不存在时  可以在后面加上 -p

  • rm

      删除文件或文件夹(remove);

  rm  -r //  删除文件夹

  • touch

      新建文件

  • cp

      拷贝文件/文件夹 (copy)
          cp -r

  • mv

      剪切文件(move)

  • clear

      清空bash屏

  • cat

      查看内容

  • vi

      vi 编辑器，对文件进行简单的编辑

  在vi编辑界面：i/a  进行编辑
              esc   退出编辑
              ：w   保存编辑文档
              ：q      退出vi界面  
              ：q!  强退

Git

  • git init 

      初始化

  • git status

      查看文件状态

  • git add . / git add name.txt 

      将文件放入暂存区

  • git commit -m 备注

      将文件上传到仓库，并添加备注到日志中

  • git log / git reflog / git log --oneline

      查看日志

  • git push xxx master

      上传文件

  • git pull xxx master

      下载文件
  • git config --global user.name

  • git config --global user.email

  • git remote add (变量名) (服务器地址)

      用变量保存服务器地址

  • git clone (远程服务器地址)

      创建文件夹+初始化+下载代码

  • touch .gitignore

      创建不受git监控的文件管理；
  在 .gitignore文件里面保存不受监控的文件的名称
